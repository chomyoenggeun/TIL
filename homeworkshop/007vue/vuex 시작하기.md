# Vuex 시작하기



### 1. 프로젝트 생성

```
$ vue create todo-vuex-app

$ cd todo-vuex-app
```





### 2. Vuex 플러그인

```
$ vue add vuex
```





### 3. Vuex-persistedstate

```
npm install --save vuex-persistedstate
```

```
import { createStore } from "vuex";
import createPersistedState from "vuex-persistedstate";
```
