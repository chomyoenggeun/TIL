# 마크다운 사용법

https://guides.github.com/features/mastering-markdown/

### 1. 제목

 문서의 구조를 맞추고 # 개수로 표현한다.

```
# 제목
## 제목
### 제목
#### 제목
##### 제목
###### 제목
```

### 2. 블록문자

 > 으로 시작하는 텍스트

```
> 블록
>> 블록
>>> 블록
```

> 블록
>
> > 블록2
> >
> > > 블록3

### 3. 목록

 숫자와 점을 사용한다.

- 순서있는 목록

```
1. 첫번째
2. 두번째
3. 세번째
1. 첫번째
2. 두번째
3. 세번째
```

- 순서없는 목록

  ```
  - 1번
  	- 2번
  		- 3번
  ```

  - 1번
    - 2번
      - 3번

### 4. 코드블록

 ``` 을 사용하여 블록을 만든다.

````
```
코드블록
````

### 5. 수평선

 아래 줄을 모두 수평선으로 만든다. 페이지 나누기 용도로 사용

```
***
------
```

------

------

### 6. 링크

```
[이름](링크)

[깃허브](www.github)
```

### 7. 강조

 강조할 때 사용

```
*강조*
_강조_
**강조**
__강조__
~~강조~~
```

*강조* *강조* **강조** **강조** ~~강조

### 8. 이미지

```
[text](경로)
```
